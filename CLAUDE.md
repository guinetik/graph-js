# Project Instructions for Claude

## Build & Dev Rules
- **DO NOT** run `npm run build`, `npm run dev`, or any build/serve commands unless explicitly asked
- Building and checking UI is the user's responsibility
- Focus on coding, testing, and code review only

## Testing
- Unit tests are welcome and encouraged
- Run tests with `npm test` when asked or when writing new test files

## Project Structure
- `graph-js/` - Core library
- `graph-js-demos/` - Vue demo application
